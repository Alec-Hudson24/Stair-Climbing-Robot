# Stair-Climbing-Robot
- The goal of this project was to design and fabricate a machine that could,
    - Be assembled in under 2 minutes and 30 seconds
    - Roll smoothly on flat ground
    - climb stairs from the bottom floor to the top and continue to roll at the top smoothly

- This project required the use of,
    - Solidworks
    - Manufactoring equipment
    - Team collaberation and organization
